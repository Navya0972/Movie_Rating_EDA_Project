**Movie Rating EDA Project**
A complete Exploratory Data Analysis on a movie ratings dataset to uncover insights about viewer preferences, rating trends, relationships between features, and patterns in audience behavior.

**Objective**
The goal of this project is to analyze a movie ratings dataset and extract meaningful insights using Python-based EDA techniques.
This analysis helps understand:
-Viewer preferences
-Genre-wise trends
-Rating distribution
-Correlations between movie features
-Patterns in audience behavior

**Dataset Information**
Name: Movie Ratings Dataset
Format: CSV
Contains: Movie names, genres, ratings, durations, votes, etc.
Purpose: Used to explore statistical patterns and visualization-based insights.

**EDA Steps Performed**
-Data loading & inspection
-Handling missing values
-Data cleaning
-Outlier detection
-Statistical summary
-Univariate analysis
-Bivariate analysis
-Correlation analysis
-Distribution plots
-Boxplots, heatmaps, countplots
-Insight extraction

**Visualizations**
All generated graphs are stored inside the graphs/ folder. Includes:
-Heatmap
-Boxplot
-Distribution plots
-Countplot
-Pairplot
-Bar charts

**Technologies & Libraries Used**
-Python
-Pandas
-NumPy
-Matplotlib
-Seaborn
-Plotly

**Installation**

1. Open notebooks/ folder

**Key Insights**
Some of the insights obtained include:
-Trends in viewer ratings
-Genre patterns
-Relationship between movie duration and rating
-Correlation between votes and rating
-Identifying common characteristics of high-rated movies
